Reporte Tcnico: Explorando Variables Explicativas de la Felicidad Mundial

1. Introduccin

El bienestar humano es un fenmeno complejo que tradicionalmente se ha intentado medir a travs del crecimiento econmico. Este reporte tiene como objetivo central cuantificar la relacin estadstica entre la riqueza de una nacin y la felicidad de sus habitantes. Para ello, utilizamos los datos ms recientes disponibles, cruzando el ndice de Felicidad de 2022 con el Producto Interno Bruto (GDP) de 2020 de ms de 140 pases.

Un desafo fundamental en este anlisis es la naturaleza exponencial de las economas nacionales: las diferencias de ingreso entre pases desarrollados y en vas de desarrollo son de rdenes de magnitud. Por esta razn, el estudio se basa en una transformacin logartmica del GDP, una decisin necesaria para corregir la asimetra de los datos y permitir una evaluacin estadstica rigurosa de su impacto real.

A lo largo del documento, contrastaremos dos enfoques: un modelo simple que asume al dinero como nico predictor, y un modelo multivariado que integra factores como la salud, el apoyo social y la libertad individual. Esta comparacin nos permitir determinar si el ingreso econmico es un fin en s mismo o simplemente un facilitador de las verdaderas causas del bienestar humano.

2. Metodologa

Para asegurar la rigurosidad estadstica y la reproducibilidad de los hallazgos, el presente anlisis se ejecut siguiendo una secuencia lgica de cuatro etapas principales:

  • Preprocesamiento y Enriquecimiento de Datos: Dado que la variable econmica original presentaba una distribucin asimtrica, se aplic una transformacin logartmica ($\ln$) al GDP para estabilizar su varianza y linealizar la tendencia. Posteriormente, se integr informacin del World Happiness Report 2022, implementando un algoritmo de limpieza de texto para homologar la nomenclatura de los pases (eliminacin de caracteres especiales) y maximizar as el tamao de la muestra final.

  • Modelado Simple (Enfoque Econmico): Se ajust un primer modelo de Regresin Lineal Simple (RLS) utilizando el mtodo de Mnimos Cuadrados Ordinarios (OLS). El objetivo de esta fase fue aislar y cuantificar el impacto del ingreso econmico sobre la felicidad, verificando a su vez los supuestos estadsticos fundamentales de normalidad (Test de Shapiro-Wilk) y homocedasticidad.

  • Modelado Mltiple (Enfoque Psicosocial): Se extendi el anlisis original incorporando tres variables predictoras adicionales: Apoyo Social, Esperanza de Vida Saludable y Libertad. Esta etapa busc evaluar cmo cambia la significancia del dinero cuando se controla por factores de bienestar humano y salud pblica.

  • Validacin y Pruebas de Hiptesis: Finalmente, se realiz una comparacin tcnica entre ambos enfoques utilizando mtricas de bondad de ajuste ($R^2$ Ajustado y Criterio AIC). De manera crtica, se aplic una Prueba de F Parcial para determinar matemticamente si existe redundancia estadstica entre el crecimiento econmico y las variables sociales.

2.2. Modelo de Regresin Lineal Simple (RLS)

Con la justificacin del EDA, procedemos a ajustar un modelo de Regresin Lineal Simple donde:

$$\text{Felicidad} = \beta_0 + \beta_1 \cdot \log(\text{GDP}) + \epsilon$$

Analizaremos los coeficientes, su significancia estadstica, y las mtricas de bondad de ajuste.

Generacin del Modelo de Regresin Lineal

Una vez validada la relacin lineal mediante la transformacin logartmica, procedemos a construir el modelo matemtico. Utilizaremos el mtodo de Mnimos Cuadrados Ordinarios (OLS).

Qu hace este cdigo? El algoritmo buscar la lnea recta "ptima" que pase lo ms cerca posible de todos los puntos. Matemticamente, calcular dos valores clave:

  • Intercepto ($\beta_0$): La felicidad base si el Log(GDP) fuera cero.
  • Pendiente ($\beta_1$): Qu tanto aumenta la felicidad por cada punto que sube el Log(GDP).

A continuacin, ejecutamos el modelo y mostramos el resumen estadstico:

Validacin del Modelo: Diagnstico de Residuos

Tener un P-value bajo no es suficiente. Para confiar plenamente en las predicciones, debemos verificar que el modelo no est cometiendo errores sistemticos.

Para esto, analizaremos los Residuos, que son la diferencia entre la felicidad real de un pas y la que nuestro modelo predijo.

                    Residuo = Felicidad Real - Felicidad Predicha

Qu vamos a hacer? Graficaremos los valores predichos (Eje X) contra los residuos (Eje Y).

  • Lo que buscamos: Una "nube" de puntos aleatoria y desordenada alrededor del cero. Esto indicara que los errores son puro azar.

  • Lo que NO queremos ver: Patrones claros como formas de "U" (que indicaran que nos falta una curva) o formas de "embudo" (que indicaran que el modelo falla ms con pases ricos que con pobres).

2.4. Modelo de Regresin Lineal Mltiple

Con el dataset consolidado, procedemos a ajustar un modelo de regresin lineal mltiple para estimar el efecto simultneo de los factores econmicos y sociales sobre la felicidad.

Ecuacin del Modelo: Planteamos la siguiente relacin lineal:

$$\text{Felicidad} = \beta_0 + \beta_1 \ln(\text{GDP}) + \beta_2 (\text{Apoyo Social}) + \beta_3 (\text{Esp. Vida}) + \beta_4 (\text{Libertad}) + \epsilon$$

Donde:

  • $\beta_0$: Intercepto (Felicidad base).

  • $\beta_1, ..., \beta_4$: Coeficientes que miden el impacto de cada variable manteniendo las dems constantes (ceteris paribus).

  • $\epsilon$: Trmino de error.

El objetivo principal es verificar si al agregar estas variables, el $R^2$ Ajustado aumenta significativamente respecto al $R^2$ Ajustado del modelo simple, lo que indicara un mayor poder explicativo.

Nota Tcnica sobre la mtrica $R^2$: Para el modelo simple anterior, utilizamos el $R^2$ estndar. Sin embargo, al agregar mltiples variables, el $R^2$ matemtico tiene una debilidad: tiende a aumentar artificialmente incluso si las nuevas variables no aportan valor real. Por lo tanto, a partir de esta seccin, utilizaremos el $R^2$ Ajustado. Esta mtrica es ms estricta, ya que "penaliza" al modelo por la complejidad aadida, permitiendo una comparacin justa y honesta sobre si las nuevas variables realmente explican mejor la felicidad.

Anlisis de Redundancia: La "Desaparicin" del Efecto del GDP

Para entender por qu el ingreso econmico perdi su significancia estadstica, realizaremos una Prueba de F Parcial (Partial F-Test).

Hiptesis: Sospechamos que existe multicolinealidad. Es decir, el dinero "compra" salud y apoyo social. Como el modelo ya tiene los datos de salud y apoyo, la informacin del dinero se vuelve redundante.

Metodologa: Compararemos el error (RSS) del modelo completo vs. un modelo reducido sin GDP. Si el error no aumenta significativamente al quitar el dinero, confirmaremos su redundancia matemtica.

In[13]:
import statsmodels.api as sm
from scipy import stats

# 1. Definir datos
y = df_final['Felicidad']

# 2. Ajustar Modelo COMPLETO (Con GDP)
X_full = df_final[['log_GDP', 'Social_Support', 'Life_Expectancy', 'Freedom']]
X_full = sm.add_constant(X_full)
modelo_full = sm.OLS(y, X_full).fit()

# 3. Ajustar Modelo REDUCIDO (Sin GDP)
X_reduced = df_final[['Social_Support', 'Life_Expectancy', 'Freedom']]
X_reduced = sm.add_constant(X_reduced)
modelo_reduced = sm.OLS(y, X_reduced).fit()

# 4. Clculo del F-Statistic Parcial
rss_full = modelo_full.ssr
rss_reduced = modelo_reduced.ssr

n = len(df_final)
p_full = 4
k = 1  # Variables removidas

numerador = (rss_reduced - rss_full) / k
denominador = rss_full / (n - p_full - 1)
f_stat_manual = numerador / denominador
p_value_manual = 1 - stats.f.cdf(f_stat_manual, k, n - p_full - 1)

# 5. Mostrar Resultados
print("="*60)
print("TEST DE REDUNDANCIA (PARTIAL F-TEST)")
print("="*60)
print(f"RSS (Error) Sin GDP: {rss_reduced:.4f}")
print(f"RSS (Error) Con GDP: {rss_full:.4f}")
print(f"Diferencia:          {rss_reduced - rss_full:.4f}")
print("-" * 60)
print(f"P-Value Calculado:   {p_value_manual:.4f}")
if p_value_manual > 0.05:
    print("--> CONCLUSIN: El GDP es estadsticamente REDUNDANTE en este modelo.")

Interpretacin Econmica y Social: Este hallazgo sugiere un fenmeno de multicolinealidad estructural y un efecto indirecto:

  1. El dinero como medio, no como fin: El ingreso nacional (GDP) es fundamental para construir sistemas de salud robustos y redes de seguridad social.

  2. Absorcin del Efecto: Una vez que el modelo incorpora las variables de resultado (Esperanza de Vida y Apoyo Social), estas capturan la varianza que originalmente explicaba el GDP.

  3. Conclusin: En este modelo, la riqueza de un pas deja de ser un predictor directo de felicidad una vez que se garantiza la salud y la libertad de sus ciudadanos. El dinero importa porque compra salud y libertad, pero el dinero por s solo no aade felicidad adicional.

3.2. Limitaciones y Recomendaciones

Para garantizar el rigor acadmico de este reporte, reconocemos las siguientes limitaciones:

  1. Correlacin $\neq$ Causalidad: Aunque variables como el Apoyo Social tienen una relacin matemtica fuerte con la felicidad, el estudio es observacional. No podemos asegurar que asignar amigos a una persona la haga feliz automticamente (podra ser una relacin bidireccional).

  2. Subjetividad Cultural: La variable objetivo es auto-reportada. Factores culturales pueden influir en cmo las personas de distintos pases califican su propia vida, independientemente de sus condiciones objetivas.

  3. Datos Agregados: Al trabajar con promedios nacionales, perdemos la visin de la desigualdad interna. Un pas con alto GDP promedio pero mal distribuido podra tener mtricas engaosas.

    Trabajo Futuro: Se sugiere incorporar el ndice de Gini en futuros modelos para evaluar el impacto de la desigualdad econmica, as como realizar un anlisis de Series de Tiempo para observar cmo evoluciona la felicidad de un mismo pas a lo largo de una dcada

Si bien el dinero (GDP) est correlacionado positivamente con la felicidad, no es el factor determinante final. Nuestro modelo demostr que el ingreso econmico acta principalmente como un facilitador.
